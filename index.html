0xMew Hax Messenger

SOCKS MODE

0xMew Hax Messenger — мессенджер с военным уровнем шифрования, работающий в реальном времени прямо в браузере.

Почему это безопасно?

Все сообщения шифруются end-to-end с помощью TweetNaCl. Даже сервер не видит содержимое ваших переписок.

Возможности

End-to-End шифрование

Ключи генерируются локально. Никто не имеет к ним доступа.

Мгновенные сообщения

Firebase Realtime Database для обмена без задержек.

Без регистрации

Никаких номеров телефонов, email или персональных данных.

Индикатор печати

Видите когда собеседник набирает сообщение.

Голосовые сообщения

Запись и отправка одним нажатием.

Голосовые звонки

Звонки прямо в чате, без видео.

Скачать для Android

Мобильное приложение

Нативный клиент для Android с полным набором функций мессенджера.

Производительность

Нативный код работает быстрее браузерной версии.

Шифрование

E2E, каждому пользователю — свой ключ.

Адаптивность

Интерфейс оптимизирован под мобильные устройства.

Как начать

  1. Перейдите во вкладку Чат
  2. Создайте комнату — получите уникальный код вида HAX-A1B2C3
  3. Поделитесь кодом с собеседником
  4. Дождитесь подключения — статус сменится на «Подключено»
  5. Общайтесь — текстом или голосом

Технологии

TweetNaCl
Криптографическая библиотека, используемая в Signal и других защищённых мессенджерах.
Firebase Realtime Database
Синхронизация сообщений в реальном времени.
Asymmetric Encryption
Публичные и приватные ключи для безопасного обмена.
Client-side only
Весь код выполняется в вашем браузере.

Важно

  • Сообщения не сохраняются после закрытия комнаты
  • Один код комнаты — одна пара собеседников
  • Не публикуйте код комнаты в открытом доступе
  • Это beta-версия — возможны баги
