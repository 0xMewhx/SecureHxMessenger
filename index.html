0xMew Hax Messenger — это не просто чат. Это полноценный мессенджер с военным уровнем шифрования, работающий в реальном времени прямо в браузере!

  1. Перейдите во вкладку 💬 Чат в меню
  2. Создайте комнату — получите уникальный код (например: HAX-A1B2C3)
  3. Поделитесь кодом с другом любым способом
  4. Дождитесь подключения — увидите статус "🟢 Подключено!"
  5. Общайтесь безопасно! Текстовые и голосовые сообщения (кнопка 🎤) 🔐

🛡️ Технологии

TweetNaCl — криптографическая библиотека, используемая в Signal и других защищённых мессенджерах.
Firebase Realtime Database — для синхронизации в реальном времени.
Asymmetric Encryption — публичные/приватные ключи для безопасного обмена.
Client-side only — весь код выполняется в вашем браузере.
